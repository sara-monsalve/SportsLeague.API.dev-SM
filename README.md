# SportsLeague API

API desarrollada en .NET 8 usando Entity Framework Core y SQL Server.

## Funcionalidades
- Gestión de equipos (Teams)
- Gestión de jugadores (Players)
- Gestión de torneos (Tournaments)
- Relaciones entre entidades

## Tecnologías
- .NET 8
- Entity Framework Core
- SQL Server
- AutoMapper

## Ejecución
1. Ejecutar migraciones
2. Ejecutar el proyecto
3. Probar endpoints en Swagger
